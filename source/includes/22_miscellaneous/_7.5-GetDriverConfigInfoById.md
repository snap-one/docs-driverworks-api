
## GetDriverConfigInfoById
Function that returns XML configuration information for a specific device.


###### Available from 4.2.0


### Signature

`C4:GetDriverConfigInfoById(num, str)`

### Parameters

| Parameter | Type | Description                                                                                                                                      |
| --------- | ---- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| DeviceId  | NUM  | Required. The ID of the device for which you want to retrieve configuration information.                                                         |
| Name      | STR  | Required. The XML tag name of the configuration item you're requesting. Note that the only XML tags that can be passed here are root level tags. |


### Returns

| Returns | Value                     |
| ------- | ------------------------- |
| data    | The value of the XML tag. |

### Example

`C4:GetDriverConfigInfoById(32, "version")`
